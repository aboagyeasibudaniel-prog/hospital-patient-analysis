# Hospital Patient Analysis Dashboard 2024

## Project Overview
A complete data analysis project performed on 500 hospital patient records using Microsoft Excel. Covers data cleaning, pivot table analysis, chart creation and dashboard design.

## Tools Used
- Microsoft Excel
- Pivot Tables
- PivotCharts

## Key Results
- Total Patients: 500
- Total Revenue: $2,044,216
- Outstanding: $503,767
- Avg Length of Stay: 7.4 days

## Key Insights
- Neurology leads with 106 patients and $466K revenue
- Corporate insurance covers the most patients (140)
- Dr. Owusu is the top doctor with $393K revenue
- July is the peak month with 59 admissions
